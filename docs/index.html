Installation

To install this add-in in PowerPoint:

  1. Download the manifest.xml file
  2. Follow the Office Add-in sideloading instructions
  3. Use the manifest URL: https://accionlabs.github.io/mermaid-powerpoint-addin/manifest.xml

  • Manifest - Add-in configuration
