📋 Installation

To install this add-in in PowerPoint or Word:

  1. Open PowerPoint or Word
  2. Go to Insert → Get Add-ins
  3. Click Upload My Add-in
  4. Select Browse and upload the manifest file, or use this URL:
https://accionlabs.github.io/mermaid-powerpoint-addin/manifest.xml

✨ Dual Platform Support
Works with both PowerPoint and Word documents
🎨 Customizable Themes
Support for multiple themes and custom colors
