üíª PowerPoint Desktop Installation

Desktop PowerPoint requires manual sideloading to specific local folders:

üìÅ Windows Instructions:

1. Download the manifest file above
2. Open File Explorer and navigate to:
%USERPROFILE%\AppData\Local\Microsoft\Office\16.0\Wef\
Create the "Wef" folder if it doesn't exist
3. Copy the downloaded manifest-production.xml file to this folder
4. Rename it to manifest.xml (remove "-production")
5. Close PowerPoint completely and restart it
6. The add-in should appear in the PowerPoint ribbon!

üçé Mac Instructions:

1. Download the manifest file above
2. Open Finder and navigate to:
~/Library/Containers/com.microsoft.PowerPoint/Data/Documents/wef
Create the "wef" folder if it doesn't exist
3. Copy the downloaded manifest-production.xml file to this folder
4. Rename it to manifest.xml (remove "-production")
5. Close PowerPoint completely and restart it
6. The add-in should appear in the PowerPoint ribbon!
üí° Troubleshooting: If you have multiple manifest files or cached versions, delete all files in the wef folder, clear the Office cache, then copy only the new manifest file.
